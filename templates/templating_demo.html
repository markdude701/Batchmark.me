Jinja Templating Examples

For Loop with Jinja

{% for o in output %}

{{o}}

{% endfor %}

If statement with Jinja

{% if output[3]|int > 10%}

{{output[3]}} is greater than 10

{% endif %}

Replace () with jinja

{{ output[0]|replace('good','Ducks') }}

Loop through string with split and make it a list.

    {% for lang in output[1].split(',') %}
  • {{lang}}
    • {% endfor %}

True HTML

{{output[2]}}|safe}}

RenderCode

    {% raw %}
        {% for cat in cats %}
            {{cat}}
        {% endfor %}
    {% endraw %}
